# 🛡️ Nexus VPN - Flutter Premium VPN Application

A modern, beautifully designed full-stack mobile application for secure and fast internet browsing. Built with Flutter, featuring a dynamic UI, real-time server updates, live connection stats, and an intuitive settings dashboard for effortless VPN management.
  
## ✨ Key Features

* **📱 Dynamic Server List:** Server screen automatically updates and creates new location chips based on the API or configuration files.
* **⚡ One-Tap Connection:** Powered by native VPN protocols for instant, secure connection without lags.
* **📊 Live Activity Logs:** Real-time data tracking showing connection duration, download, and upload speeds.
* **⚙️ Advanced Settings:** Built-in dashboard to easily modify VPN protocols, app themes, and notifications directly from the app.
* **🌍 Diverse Locations:** Supports various server locations across multiple regions for unrestricted browsing.
* **🚀 Smooth Animations & UI:** Clean, responsive, and modern Material design tailored for the best user experience.

## 🛠️ Tech Stack & Packages

This project utilizes the following core technologies and packages:
* **`flutter`** - UI Toolkit
* **`openvpn_flutter`** - For handling OpenVPN profiles and connections (or Custom Method Channels)
* **`get`** / **`provider`** - For efficient state management
* **`shared_preferences`** - For saving user settings and preferences locally

## 🚀 Getting Started

Follow these steps to run the project on your local machine.

### Prerequisites

* Flutter SDK (v3.10.0 or higher)
* Android Studio / VS Code
* OpenVPN Configuration files (`.ovpn`)
